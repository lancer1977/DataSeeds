# Project Atlas

## Purpose

`DataSeeds` contains data conversion utilities and seed-data support assets.

## Entry points

- `CsvToJson/CsvToJson.sln` - current CI build entry point.
- `PFAExcelToXML/PFAExcelToXML.sln` - legacy .NET Framework conversion lane.
- `docs/features/` - generated feature index and repo-shape notes.
- `scripts/validate.sh` - CI validation entry point.
